# Hero Images Folder

Drop your images here to display them on the main page instead of text boxes.

## Supported Formats
- JPG/JPEG
- PNG
- SVG
- WebP

## Naming Convention
Images will be displayed in alphabetical order by filename.

Example filenames:
- 01-robotics-lab.jpg
- 02-autonomous-vehicle.png
- 03-medical-device.jpg
- 04-production-system.png

## Recommended Size
- Width: 400-600px
- Height: 300-400px
- Keep file sizes under 500KB for optimal loading
